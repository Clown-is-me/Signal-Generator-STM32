Цифровая симуляция аналоговых сигналов на STM32F103RB c дополнительным модулем Multi-function Shield

По одиночному нажатию на тактовую кнопку А1 сгенерировать сигнал, амплитуда частоты которого находится в диапазоне [-1; 1], период 2 с.  На 4-сегментовый дисплей выводить текущее значение сигнала.
По одиночному нажатию на тактовую кнопку А2 увеличить амплитуду полученного сигнала на 1 до 3. Амплитуда изменяется циклично.
По одиночному нажатию на тактовую кнопку А3 сгенерировать случайный сигнал, составить полифонический с основным.
